CSS Backgrounds

The CSS background properties are used to add background effects for elements.

In these chapters, you will learn about the following CSS background properties:

  • background-color
  • background-image
  • background-attachment
  • background-position
  • background (shorthand property)

background-color

The background-color property sets the background color of an element. You can use color names, hex codes, RGB, or HSL values. It helps make sections of your page visually distinct. For example, you can set background-color: lightblue; for a div. It only applies to the element's content area by default.

background-image

The background-image property sets an image as the background of an element. You can use images from your computer or online URLs. Example: background-image: url('image.jpg');. It can be combined with other background properties for better design. Images can enhance the visual appeal of your web page.

background-repeat

The background-repeat property controls whether a background image repeats. By default, it repeats both horizontally and vertically. You can use repeat-x, repeat-y, or no-repeat. It is useful when you want to control how a pattern or image appears. Proper use helps prevent distorted or crowded backgrounds.

background-attachment

The background-attachment property sets whether a background image scrolls with the page. The default value is scroll, meaning the image moves as you scroll. Using fixed keeps the background in place while content scrolls over it. This property is often used for parallax effects. It enhances the depth and feel of your web page.

background-position

The background-position property determines where a background image is placed. You can set it using keywords like top, center, bottom, or with coordinates. Example: background-position: right bottom;. It helps align images exactly how you want within an element. Combined with other properties, it gives precise control over layout.

background (shorthand property)

The background property is a shorthand to set multiple background properties in one line. It can include color, image, repeat, attachment, and position. Example: background: lightblue url('image.jpg') no-repeat center fixed;. Using shorthand makes your CSS cleaner and easier to manage. It is convenient for setting a complete background style quickly.
