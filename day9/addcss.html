How to Add CSS

When a browser reads a style sheet, it will format the HTML document according to the information in the style sheet.

There are three ways of inserting a style sheet:

  • External CSS
  • Internal CSS
  • Inline CSS

External CSS

External CSS is written in a separate .css file and linked to the HTML document using the <link> tag inside the <head> section. This method allows you to use the same CSS file for multiple HTML pages, making your website easier to maintain. It keeps HTML and CSS separate, which is a good practice for larger websites. Example: <link rel="stylesheet" href="style.css">.

Internal CSS

Internal CSS is written inside a <style> tag within the <head> section of the HTML document. This method is useful for styling a single page or for testing small changes. It does not require a separate file, so all styles are contained within the HTML file. However, it is less efficient for larger websites because the styles cannot be reused across multiple pages.

Inline CSS

Inline CSS is added directly to an HTML element using the style attribute. This method allows you to apply styles to a specific element only. Example: <p style="color: blue; font-size: 16px;">Hello World</p>. Inline CSS is useful for quick changes or overriding other styles, but it is not recommended for maintaining a clean and organized code structure.
